Como usarlo 

En la carpeta raiz busque los .exe y ejecutelos ahi vera el flujo de cada uno en su respectivo lenguaje 
en el caso que no esten los puntos .exe ejecutar por terminal

Ejecucion por terminal:
C++:
-g++ ejemplo1.cpp -o ejemplo1.exe
Haskell:
-ghc ejemplo2.hs -o ejemplo2.exe
Rust:
rustc ejemplo3.rs -o ejemplo3.exe
